# Pi Distribution Governance Adoption

This repository adopts AI-Native Engineering Operating System **2.2** from
`pcvantol/djconnect/docs/governance/PLATFORM_ARCHITECT_SYSTEM_INSTRUCTIONS.md`.
There are no local exceptions.

Definition of Done: synchronized clean main, artifact manifest/checksum and
release-note validation, updated rolling records/immutable history and one
reviewable PR. Pi source and service runtime remain outside this repository.

Release profile: Pi packages/bundles, checksums, manifests and public release
notes. Docker, Cloudflare Workers and HACS are not targets.
